About Python

Python is a popular programming language created by Guido van Rossum. It is known for being easy to read and simple to use.

History

1989 - Python is Born

Guido van Rossum started working on Python in 1989 while at CWI in the Netherlands.

1991 - First Release

Python 1.0 was released in 1991 and quickly gained attention for its simple design.

Today

Python is now one of the most popular programming languages used around the world.
